Bygginstruktioner i terminalen:
Klona repository från Github-länken till valfri mapp
Skriv: cd FoodOrder för att komma till mappen
Skriv: mvn clean package
Skriv: ls, en ny katalog ska nu finnas som heter target
Skriv: cd target
Skriv: ls
En fil vid namn: FoodOrder-1.0-SNAPSHOT.jar ska nu finnas - kopiera den
Skriv: java -jar och klistra in: FoodOrder-1.0-SNAPSHOT.jar
Det ska se ut såhär: java -jar FoodOrder-1.0-SNAPSHOT.jar
Och nu ska programmet köra!
